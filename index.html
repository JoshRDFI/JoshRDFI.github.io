      This project explores how to create a supervised machine learning model to determine which people would make good donors for a charity. The data is provided and comes in a CSV file derived from the UCI Machine Learning Repository.

  • Udacity R Programming Bicycle Data Exploration

    This course taught programming and visualizations in R Studio using GGPlot2. The course provided bicycle rental csv files that informed the 3 questions I had to develop and answer, both through visualizations and a statistical summary.
    Certificate of Completion

  • Airport Delays Project on Tableau Public

    In this course, I learned how to use Tableau (Public) and created a Tableau dashboard to explore flight data provided within the course. The data was provided in 3 CSV files and I had to determine what questions to ask. These questions would inform the creation of the dashboard.
    Certificate of Completion

  • Udacity Data Modeling With Postgres

    In this project a database of all the songs and artists played by users on the Sparkify platform was created with ETL processes. The database software chosen was PostgreSQL. Python was chosen as the language to extract the data from the JSON files and import it into the database. Additionally, Python is used to create the database and its associated tables.
    Certificate of Completion

    Certificate of Completion

    • Udacity Investigate a Dataset

    • Udacity A/B Testing
